## 💻 Começando

<h3>
  Algoritmo
</h3>

**Clone o projeto e acesse a pasta**

```bash
$ git clone https://github.com/LuizSouz404/Cowala-Challenger.git && cd algoritmo
```

<h3>
  ReactJS
</h3>

```bash
$ cd.. && cd frontend
```

**Siga os passos abaixo**
```bash
# Instalar as depedências
$ yarn

# Iniciar o projeto ReactJS
$ yarn start
```
